### Setup Requirements for Production

1. **Vercel Account** (for frontend)
   - Free tier available
   - Deploy Next.js apps
   - Auto SSL certificates
   - Custom domain support

2. **DigitalOcean Droplet** (for backend)
   - $6/month minimum
   - Ubuntu 22.04 LTS recommended
   - 2GB RAM sufficient for startup
   - Scalable as needed

3. **PostgreSQL Database** (choose one)
   - Neon (free tier available, PostgreSQL compatible)
   - DigitalOcean Managed Database ($15/month)
   - Self-hosted (included in Droplet)

4. **Custom Domain** (optional but recommended)
   - Point frontend to Vercel
   - Point API to DigitalOcean
   - Cloudflare for DNS (free tier available)

5. **Email Service** (for notifications)
   - Cloudflare Email Routing (free)
   - SendGrid, Mailgun, or similar
   - Configured in backend .env
